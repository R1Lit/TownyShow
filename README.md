# TownyShow

Plugin for Towny that adds customisation of notifications to the actionbar/titles.

**IMPORTANT! Set the value** `notifications_appear_as: none` **in towny config.yml.**

## Requirements

* **Spigot(maybe) / Paper** 1.20+
* **Towny**
* **PlaceholderAPI**

---

## Placeholders

#### Town

* `%townyshow_town_name%`
* `%townyshow_town_nation%`

#### Plot

* `%townyshow_plot_name%`
* `%townyshow_plot_owner%`
* `%townyshow_plot_price%`
* `%townyshow_plot_home%`
